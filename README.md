# Tickety App

## Installation

```bash
git clone https://github.com/drzioner/tickety.git

cd tickety

```

## Execution

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

First, run the production server:

```bash
npm run start
# or
yarn start
```
